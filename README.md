## 产品定位
企业人力资源管理系统

## 产品解决的痛点
解决中小企业，人力资源管理的问题

## 产品潜在客户
1. 中小企业

## 产品特色
1. 微信请假
2. 企业认证中心，其他CRM，资料管理系统，财务管理系统，都由EHR统一认证。

## 需求描述
1. 员工级别信息管理
2. 员工入职管理
3. 员工奖惩管理
4. 员工转岗管理
5. 员工离职管理等
6. 员工请假管理
7. 企业认证中心
8. ....

## 后续版本功能
加班管理
报销管理
员工薪酬管理

## 技术架构
1. Java：Nutz框架
2. 缓存：EhCache，
3. 静态模板框架：Beetl
4. 后台UI框架：B-JUI
5. 数据库：MySql
6. 后台服务器：阿里云centos6
